Welcome to GLP1 Medical Assistant

About this Service

This is a specialized medical information assistant focused on GLP-1 medications. The service provides:

  • Evidence-based information about GLP-1 medications
  • Details about proper usage and administration
  • Information about potential side effects
  • Storage requirements and guidelines
  • Lifestyle modification recommendations

Note: This is an API service. To interact with the assistant, please use the /api/chat endpoint.
